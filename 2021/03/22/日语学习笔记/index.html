第一课

AはBです A是B(表肯定)
AはBですか A是B吗(表疑问，か升调)
AはBでは、ありません A不是B(表否定)
名词和名词之间不能直接连接，要用の连接

第二课

これ(这个 距离近)　それ(那个 距离稍远)　あれ(这个 距离很远) どれ(哪个)

第三课

ここ(这里 距离近)　そこ(那里 稍远)　あそこ(那里 距离很远)　どこ(哪里)
对应的更礼貌的说法
こちら　　　　　　　そちら　　　　　　あちら　　　　　　　　　どちら
月火水木金土日(周 一二三四五六七)

第四课

  1. 东西/人　は 地点　に　あります/います 这里动物和人用います ，其他用あります
  2. 地点 に 东西/人　が　ある/いる 这里动物和人用いる，其他用ある
  3. ま　助词 表示也的意思
  4. と　助词 表示和的意思 表达A和B的时候会用 A　と　B

    第五课

きのう 昨天
きょう 今天
あした 明天

あさ　早
ひる　中
ばん　晚
よる　夜

具体时间 + に + 动词
表示动作发生的具体时间点

ます　（现在/将来时 肯定)
ません (现在/将来时 否定)
ました （过去时 肯定）
ませんでした (过去时 否定)

から…まで 从…到…
感觉可以理解为from … to…

第六课

地方 + へ 表示移动的方向。这里へ读作e
交通工具+で 表示乘坐某种交通工具

第七课

これから　从今往后
something+を+动词 做某事(有点像英语的被动式)
地点+で+动词 动作发生的地点
A か B A或B

もの　＋　を　＋　ください
请给我xxx

第八课

工具 + で
使用某种工具/方法/途径

物品授受关系

A给B 给：あげる　
给的后面不能是“我”
我给他，他给他都用 あげる
收到别人给的东西用 もらう

会面

人 + に　+ 会います
与某个人见面
