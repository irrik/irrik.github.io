无线网络概述

组成

无线网络由一个AP(access point)和一个或一个以上的STA(station)组成。通常将无线路由器称为AP，
接入的客户端称为STA.

工作流程

首先要解释一下。AP会定时地广播SSID，所以STA可以监听到AP发出的信号。
通俗点来说，无线路由器会定时发出广播信号，手机电脑等在收到信号后可以感知到wifi的存在。
工作流程大致如下

  1. 无线路由器定时广播SSID，STA尝试加入无线网络时会发送一个探测请求。AP收到该请求后会返回一个含频带信息的响应包，此时，STA会切换到指定频带。
  2. STA将提供认证密码，如果密码正确，AP将允许STA接入无线网络。
  3. SAT和AP建立关联，在关联过程中，STA和AP之间要根据信号的强弱协商速率，直至关联成功
  4. STA和AP开始进行数据的收发。

    重要的部分在于频段的划分，可以看到这些标准协议用的都是2.4Ghz或者5Ghz的频段。分别讲一下这两有什么不同。

    2.4Ghz频段

频段指的是无线信号的频率范围。无线信号在规定的频率范围内传输数据。
2.4Ghz的频率范围是2.4Ghz~2.4835Ghz
为了充分利用该频段，将该范围划分为了几个部分，每个部分称为一个信道。
目前，主流的wifi网络一般都支持13个信道。他们的中心频率不同，但是，因为都占据一定的频率范围，所以会有些互相重叠的情况。
13个信道的频率范围如图所示：

图中也有写道，有三个不互相重叠的信道。
分别是1，6，11这三个信道。
每个信道的有效范围是20Hz，但是实践占了22Hz，因为要留2Hz的强制隔离频带。

带宽

这部分也是比较有意思的，解释了带宽和网速有关哈哈。
这里的带宽指的是信道带宽。
在常用的2.4~2.4835Ghz频段上，每个信道的带宽为20Mhz，在802.11n模式下能达到144Mbps带宽，但是穿透性比较好。
在40Mhz且802.11n模式下，能达到300Mbps的带宽.
不难理解，使用的频段带宽越大，传输速率越大。但是，频段是大家共享的，如果你使用了过大的频段而导致信号干扰，为了对抗干扰，无线会降速。最终将导致得不偿失
但是。如果你家周围路由器比较少，可以尝试将路由器工作的频段调大，这将会使速度得到提升(亲测有用)

无线网络安全保障

感觉这部分讲得有点啰嗦，就总结一点感觉有用的东西就行。
要连接到路由器，一般都要输入密码的。大部分路由器支持以下三种无线加密方式
WEP,WPS,WPA/WPA2.
不加密的方式就不说了感觉好，，弱智。

WEP

这种方式有严重的安全漏洞，可以不需要密码字典，直接碰撞得出。不建议使用

WPA/WPA2

这种加密方法很安全，但是还是可以通过抓握手包加跑字典的方式破解。嗯，前提是有个好字典。
这种加密有两种加密模式，WPA-PSK/WPA2-PSK一般用于小型网络或者家庭网络。WPA/WPA2一般用于大型企业
WPS就不讲了。因为没搞懂。
除了连接过程的加密，信息也有加密的算法

无线网络监听

无线网络的数据包是以无线信号的方式传播的，所以可以对该网络的数据包进行监听。(你看谍战剧不就有这种情节吗，拦截敌方的无线电然后解密)
要对无线网络进行监听，必须将无线网卡设置为监听模式
简单讲一下无线网卡的工作模式

  1. 被管理模式：就是正常当网卡连接基站上网。
  2. Ad hoc模式：点对点模式，就直接无线客户端到无线客户端了。
  3. 主模式：就是把网卡作为AP，允许其他设备接入
  4. 监听模式(monitor mode)：从使用角度来说，这是我们最关心的模式，也是最重要的模式。

有时候，会出现明明网卡成功接入，也成功开启监听模式但是监听不到数据的情况
我自己体会的话，总结了两种方法吧。
一是先开虚拟机再插入无线网卡，然后把网卡连接到虚拟机，这样就可以解决。
二是如果先插入网卡的话，将网卡连接到虚拟机后，拔出网卡，换个usb口再插入就行。
